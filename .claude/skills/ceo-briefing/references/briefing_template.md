# CEO Briefing Template Reference

Standard template for Monday Morning CEO Briefing.

## Required Sections

1. **Executive Summary** - 2-3 sentences max
2. **Revenue** - Week, MTD, trend
3. **Completed Tasks** - Checkbox list
4. **Bottlenecks** - Table format
5. **Proactive Suggestions** - Actionable items
6. **Upcoming Deadlines** - Next 7 days

## Metrics to Track

| Metric | Source | Calculation |
|--------|--------|-------------|
| Revenue | Accounting/ | Sum of invoices |
| Tasks Done | Done/ folder | File count |
| Emails Sent | Logs/ | Action count |
| Response Time | Logs/ | Avg time to reply |

## Tone Guidelines

- Direct and factual
- No fluff or pleasantries
- Actionable recommendations
- Data-backed insights

## Example Proactive Suggestions

### Cost Optimization
- "Notion: No activity in 45 days. Cost: $15/month. Consider canceling?"
- "Adobe: Usage dropped 60%. Downgrade plan?"

### Revenue Opportunities
- "Client X: Last invoice 60 days ago. Follow up?"
- "Project Y: 80% complete. Invoice milestone?"

### Risk Alerts
- "Deadline: Project Z due in 3 days"
- "Payment: Invoice #123 overdue 15 days"
